# WEDDING DRESS CODE

A deploy-ready Next.js, React, and Tailwind CSS landing page for Shaneika & George's Wedding Tea Party dress code.

## Local Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
```

## Deploy to Vercel

Import the GitHub repository in Vercel, or run:

```bash
npm i -g vercel
vercel --prod
```

The app uses local image assets from `public/images/dress-code/`, optimized with the Next.js `Image` component.
